# 🌟 Vision Transformer Image Classifier

An AI-powered image classification web application built using **Vision Transformer (ViT)**, **PyTorch**, **Hugging Face Transformers**, and **Gradio**.  
Upload any image and let the AI predict the object with confidence scores in real time.

## 🚀 Features

✔️ Upload and classify any image  
✔️ Vision Transformer (ViT) based prediction  
✔️ Confidence score display  
✔️ Interactive Gradio interface  
✔️ Fast and user-friendly UI
